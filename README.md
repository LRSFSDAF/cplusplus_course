# C++ Programming Lab Experiments (C++ 课程实验作业)

欢迎来到我的 C++ 课程实验代码仓库。本仓库包含了四次阶段性实验的完整代码、题目分析及运行结果截图。

本项目使用 **VS Code** 进行开发，代码涵盖了从 C++ 基础语法到面向对象编程（OOP）的各个知识点。

## 📂 实验目录概览

点击下方的链接可进入对应的实验文件夹查看详细题目与代码。

| 实验编号 | 主题 | 包含题目摘要 | 链接 |
| :--- | :--- | :--- | :--- |
| **实验一** | **C++ 基础编程** | 一元二次方程、倍数判断、大小写转换、字符统计 | [进入目录](./Experiment_01/) |
| **实验二** | **数据结构与指针** | 十进制转二进制、二维向量指针操作、回文判断、约瑟夫问题 | [进入目录](./Experiment_02/) |
| **实验三** | **函数、递归与作用域** | 迭代法求平方根、变量作用域分析、字符串加密、勒让德多项式(递归) | [进入目录](./Experiment_03/) |
| **实验四** | **面向对象编程 (OOP)** | 矩形类定义、复数类定义、运算符重载、集合类(Set)封装 | [进入目录](./Experiment_04/) |
